⚙️ 配置面板

在网页上直接管理监控航线，保存后写回仓库配置并可一键触发抓取。

1. GitHub 访问令牌（Token）

本面板通过 GitHub API 读写你的仓库配置。需要一个 fine-grained personal access token：
  1. 打开 GitHub → 右上角头像 → Settings → 左下 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token。
  2. Repository access 选 Only select repositories，只勾选 cheapest-flights（不要授权全部仓库）。
  3. Permissions → Repository permissions：Contents 设为 Read and write；Actions 设为 Read and write（用于「立即抓取」触发工作流）。
  4. 生成后复制 token（形如 github_pat_...），粘贴到下面。Token 仅保存在本浏览器 localStorage，不会上传任何服务器。

2. 监控航线

从仓库的 config.json 读取当前航线。编辑 / 删除 / 启停后需点「保存全部更改」写回仓库。

新增航线

填英文全名，如 China Eastern。抓取源返回的是航司全名而非二字码，回车或逗号确认添加。

3. 保存并抓取

「保存全部更改」会同时写回 config.json 与 config.yaml（保持两者同步，供 src/config.py 解析）。保存成功后可点「立即抓取」触发 GitHub Actions。
